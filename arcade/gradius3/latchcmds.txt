ym2151 voice commands:
e0: key off, set callback7 flag to op1
e1: set vstate 10 to op1
e2: patch change (FM) set FM props from table[op1]
e3: set level adjustment 1 to op1
e4: set vstate 12 to op1
e5: utility duration:  zero duration; set fc1e from op1, add op1 to duration;
    while (opN nybble not zero) add high nybble duration;
    add low nybble duration
e6: set voice fine tuning from op1
e7: set set-keycode flag from op1
e8: set global transpose op1.6-4 octave op1.3-0 semis op1.7 sign
e9: set per-voice transpose from op1
ea: set global level adj from op1
eb: set pervoice level adj 2 from op1
ec: dec vstate 1e if 0 or op1 == 1; if op2 nonzero inc fc26 and add transpose semis from op2 to field 1c; if op3 nonzero inc fc26 and add abs(op3) to 1d; goto op4/5
ed: set lfo params: op1: lfo_freq op2:lfo PM depth op3:lfo AM depth op4: lfo waveform
ee: set vstate 36 from op1 (37 is reset to this at note start)
ef: set pan flags
f0: goto  op1/2: dest addr LH
f1: begin(?) repeat,  op1: rpt count op 2/3:dest on last rpt L/H
f2: mark begin repeat, op1: rpt count
f3: end f2 repeat
f4: mark begin repeat, op1: rpt count
f5: end f4 repeat
f6: call subroutine, op1: repeat count, op 2/3: dest
f7: return from subroutine
f8: begin(?) repeat,  op1: rpt count op 2/3:dest on last rpt L/H
